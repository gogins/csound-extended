Introduction

The purpose of this demonstration is to show how to embed a Csound piece into a Web page. You can play the piece, stop it from playing, view and edit its Csound code, and see Csound's diagnostic messages. This piece will play on csound.node.

Csound Orchestra

How It Is Done

This demonstration is simplified. All unnecessary HTML elements and JavaScript function calls have been omitted. This Web page uses only a subset of the Csound API that has the same funtions, names, and signatures in csound.node, Csound for Android, and CsoundQt.

Comments in the page source explain what each element is for and why it is written the way it is. You can view the code and comments by right-clicking on this page and selecting "View page source".
